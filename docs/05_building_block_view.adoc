== Klassendiagramm
Das nachfolgende Diagramm zeigt die Klassen und ihre Methoden des Simulators.


[plantuml, Sequenzdiagramm, svg]
----
@startuml
skinparam classAttributeIconSize 0

class EdgeDetection {
  +load()
  +find_contours()
  +filter_contours()
  +get_puzzle_pieces()
}

class PuzzlePiece {
  +get_puzzle_edges()
  +rotate()
  +translate()
}

class PuzzleEdge {
  +edge_type
  +points
}

class Matching {
  +find_matches(threshold)
}

class Match {
  +score
  +edge_a
  +edge_b
}

class PuzzleOrganizer {
  +organize()
}

class FlatEdgeFinder {
  +log_edge_types()
}

class Anchor {
  +choose_anchor()
  +place_anchor()
}

class Rotation {
  +rotate()
}

class Translation {
  +translate()
}

class MatchPlacer {
  +apply_matches()
}

class GlobalArea {
  +set_unsolved_puzzles()
  +set_solved_puzzles()
  +scale_all_puzzles()
  +translate_unsolved_puzzles()
  +show()
}

EdgeDetection --> PuzzlePiece : erzeugt

MatchPlacer --> Rotation
MatchPlacer --> Translation

GlobalArea --> PuzzlePiece

@enduml
